# 进销存管理系统-Java
### 1、介绍
本系统围绕进销存管理中的核心业务流程进行设计，功能全面且流程清晰， 后端：Java 21 + Spring Boot 2.7，前端：Vue 2 + Element UI，数据库：MySQL 8.0，源码可用于编程实战练习~
